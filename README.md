# 🚀 Resume Analyzer (Frontend)

This is a simple frontend-only Resume Analyzer built using React.

## 📌 Features

- Upload Resume (PDF only)
- Mocked Resume Analysis
- Score Generation (0–100)
- Visual Progress Bar
- Improvement Suggestions

## 🛠️ Tech Stack

- React (Functional Components)
- JavaScript
- CSS

## ⚙️ How It Works

1. User uploads a resume (PDF file)
2. App simulates keyword analysis (mocked logic)
3. A random score is generated
4. Suggestions are shown based on the score

```bash
npm install
npm run dev
